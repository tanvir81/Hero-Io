HERO.IO

A modern web platform to explore, install, and manage trending apps with a clean UI and responsive design.

 Features
- Search and filter apps by title
- View app details and installation status
- Custom 404 and error pages
- Responsive layout with Tailwind CSS
- Smooth UX with debounced search

#Technologies Used
- React
- Vite
- Tailwind CSS
- DaisyUI
- React Router
- Netlify (for deployment)
