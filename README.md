HERO.IO

A modern web platform to explore, install, and manage trending apps with a clean UI and responsive design.

# Features

1. Search and filter apps by title
2. View app details and installation status
3. Custom 404 and error pages
4. Responsive layout with Tailwind CSS
5. Smooth UX with debounced search

# Technologies Used

1. React
2. Vite
3. Tailwind CSS
4. DaisyUI
5. React Router
6. Netlify (for deployment)
